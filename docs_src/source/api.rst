`API` class documentation
=====================================
.. py:module:: apreshttp

Classes related to the top-level API entry point are described here.  The :py:class:`API` class is instantiated to access subsequent API methods and provide the same hierarchy as the REST API describes.  The :py:class:`APIChild` class is used to describe
common methods for entry points into the API with methods to expose GET and POST requests.
